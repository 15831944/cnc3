Задание смещения проволоки и режимов работы станка

Левое смещение

При резе используется левое смещение проволоки относительно контура по направлению реза.

Правое смещение

При резе используется правое смещение проволоки относительно контура по направлению реза.

Количество проходов

Число проходов по контуру реза. Число от 1 до 7.

Смещение при проходе 1..7

Смещение проволоки относительно контура реза на 1, 2 и т.д. проходах.

Режим при проходе 1..7

Режим работы станка на 1, 2 и т.д. проходах.

Режим линии захода

Режим работы станка при движение по линии захода.

Недорез

Длина недореза контура детали. Используется для того, чтобы не выпадала деталь при многопроходном резе замкнутого контура.

Смещение при дорезе

Смещение проволоки при дорезе (при однопроходном дорезе).

Режим при дорезе

Режим работы станка при дорезе (при однопроходном дорезе).

Остановка перед дорезанием

Остановить программу перед дорезанием контура.

Перерез

Задание длины перереза в мм.

Задержка по включению насоса

Вставляет задержку выполнения программы по включению насоса в секундах.

Остановка по включению насоса

Остановка программы после включения насоса и заданной задержки.

  • Новый - добавить новый режим

  • Удалить - удалить выбранный режим

  • Открыть - открыть файл с режимами работы

  • Сохранить как - сохранить режимы работы как новый файл
