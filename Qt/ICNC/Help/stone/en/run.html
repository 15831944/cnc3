Work panel

Buttons

  • Home - Go to the Home panel

    The button is disabled during running.

  • Break - Wire break control turn On/Off

  • Pump - Cutting fluid turn On/Off

  • Drum - Wire drum turn On/Off

  • Start/Stop

    Start - start or continue cutting program;
    Stop - stop cutting program.

  • Back - Return back along the cut contour

    This button is active after pressing the "Stop" button. M and G-codes that control the operation modes are ignored when CNC is reading the program in the reverse direction.

  • Cancel - Cancel cutting

    The button is active after pressing the "Stop" button. By pressing this button, the current state of the CNC is saved and the cut is turned off. After turning off the power, you can resume work from the same place using the "Recovery" button on the Home panel.

  • Hold - Motor holding turn On/Off

    Enable/disable motor drivers. When the motor drivers are enabled, the motors are holding the workpiece otherwise you can move it manually.

Controls

  • Drum - Drum velocity

    Value range [1, 7].

  • Speed - Cutting speed

    Value range [0.01, 18] мм/мин.

Flag

  • Idle run

    Enable/disable idle mode. In idle mode, the CNC is allowed to work in modes that are not compatible with cutting a workpiece. The idle mode is used for tuning purposes.

State panel

The state panel displays:

  • Current coordinates X, Y in millimeters;
  • Elapsed time from the start in HH:MM:SS format;
  • Estimated remaining time in HH:MM:SS format.
  • Flags:
    • PWR - no power supply 220V;
    • WB - a wire break signal;
    • ALM - an alarm signal from one of the limit switches;
    • REV - the limit switch of the backward motion of the drum is active;
    • FWD - the limit switch of the forward motion of the drum is active.
