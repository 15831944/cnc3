Application settings

  • Language

    Choosing the language of the program interface. The changes take effect after a reboot.

  • Reverse plot axis X

    Axis X will be reversed only on the screen

  • Reverse plot axis Y

    Axis Y will be reversed only on the screen

  • Swap plot axes X, Y

    Axes will be swapped only on the screen

  • Show plot axes names

    Show plot axes names on the screen

CNC settings

  • Input levels

    From the drop-down menu, you can select preset CNC input levels for EDM (Metal) or stone cutting (Stone), or program debugging (Debug).

  • Calculation step

    Calculation step of the next position.

  • Reverse motor X

    Reverse a motor direction for axis X.

  • Reverse motor Y

    Reverse a motor direction for axis Y.

  • Swap motors X and Y

    Swap axes X and Y for the machine bed.

  • Motor precisions

    A number of motor steps per mm for each axis.

Buttons

  • Home - Go to the Home panel

  • Read - Read CNC settings

  • Write - Write CNC settings

  • Default - Set default settings
