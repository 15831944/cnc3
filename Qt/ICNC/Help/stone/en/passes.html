Contour Adjustments

  • Wire offset

    Wire width compensation. This parameter changes the contour to compensate for the width of the cutting wire.

    • Left Offset

      Use wire offset to the left of the cutting direction.

    • Right Offset

      Use wire offset to the right of the cutting direction.

  • Additional Offset

    An additional offset can be used to resize the contour, but contour proportions might be distorted.

  • Tab Width

    The CNC machine can stop before it reaches the end of the contour, leaving a tab (or bridge) to hold a part in the workpiece. It prevents a part from falling down. After clamping the part, you can continue cutting by pressing the "Start" button.

  • Overlap

    This parameter sets the length of the contour overcut. It's used in conjunction with a tab to process an entry point.

  • Cutting Speed

    Setting a cutting speed in mm/min.

  • Pause

    Adding a pause after each contour segment.

Buttons

  • Back

    Return to the Contour Editor.

  • Default

    Set default values.
