Задание смещения проволоки и режимов работы станка

Смещение проволоки

Смещение проволоки относительно контура реза.

Левое смещение

При резе используется левое смещение проволоки относительно контура по направлению реза.

Правое смещение

При резе используется правое смещение проволоки относительно контура по направлению реза.

Дополнительное смещение

Смещение изменяет размер контура, пропорции при этом могут исказиться.

Недорез

Задание длины недореза контура. Станок автоматически останавливается недорезав контур, для продолжения реза требуется повторный запуск. Недорез используется для того, чтобы деталь не выпала по завершению реза замкнутого контура.

Перерез

Задание длины перереза контура. Используется совмество с недорезом для обработки точки входа.

Скорость реза

Задание скорости реза в миллиметрах в минуту.

Паузы

Добавление паузы в заданное количество секунд после каждого сегмента контура.
