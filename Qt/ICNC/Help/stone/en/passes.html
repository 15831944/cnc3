Adjustments

Wire offset

Wire width compensation. Wire offset relative to the given contour.

Left Offset

Use wire offset to the left of the cutting direction.

Right Offset

Use wire offset to the right of the cutting direction.

Additional Offset

The additional offset can be used to resize the contour, but contour proportions may be distorted.

Tab Width

The parameter sets the length of the contour undercut. The machine will stop before it reaches the end of the contour, leaving the set length (tab width). It is used to prevent the part from falling out after cutting is completed.

Overlap

The parameter sets the length of the contour overcut. It is used in conjunction with undercutting to process the starting point.

Cutting Speed

Setting the cutting speed in millimeters per minute.

Pause

Adding a pause after each contour segment.
