G-code Editor

Buttons

  • Home - go to the home panel

  • New - create a new G-code file

  • Open - open a G-code file

    Open a file with a CNC program in G-code format. The file extension should be (*.NC).

  • Save

        Save current G-code file.

  • Save as

    Save G-code with a new name

  • Plot

    Show the contour that is described in G-code.

  • to Contour

    Converting a G-code to a contour to further customize the contour processing parameters.

  • Run

    Go to the Run panel with the current G-code program.

Supported G-code commands

Control character

% - delimiter

It is set at the beginning and the end of the G-code program.

Comments

({text of comment})

Example:
G1 X10 Y0 (cutline)

G-codes

G0 X{x} Y{y} - Rapid positioning

Moving to a point (x,y) with maximum speed by linear trajectory. x, y - are given in mm.

G1 X{x} Y{y} F{speed} - Linear interpolation

Moving to a point (x,y) with a given speed by linear trajectory. x, y - are given in mm, speed - in mm/min, [0.01, 18].

G2 X{x} Y{y} I{Cx} J{Cy} F{speed} - Circular interpolation, clockwise

G2 X{x} Y{y} R{radius} F{speed} - Circular interpolation, clockwise

G3 X{x} Y{y} I{Cx} J{Cy} F{speed} - Circular interpolation, counterclockwise

G3 X{x} Y{y} R{radius} F{speed} - Circular interpolation, counterclockwise

G4 P{pause} - Pause, milliseconds

The program pauses for the given time.

Example:
M40 (- pump enable)
G4 P30000 (- pause for 30 seconds)
M82 (- drum enable)

G90 - Absolute programming (by default)

G91 - Incremental programming

G92 X{x} Y{y} - Position register


Modal Codes

Modal codes G1, G2, G3, X, Y, F remain in effect until replaced by a contradictory code.
Example:
%
G92 X0 Y0
G1 X10 Y20 F5
X20 Y0
X0
M2
%

M-codes

M0 - Compulsory stop

The G-code program halts and waits for the Start button to be pressed.

Example:
M40 (- pump enable)
G4 P30000 (- pause 30 for seconds)
M0 (- halt the program)
M82 (- drum enable)

M2, M30 - End of program

You can use any of the commands to end the program.

M40 - Pump enable

M41 - Pump disable

M82 - Drum enable

M83 - Drum disable
