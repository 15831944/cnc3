Contour Editor

Buttons

Home - go to the Home panel

Load DXF - load DXF file

Load a cut contour in DXF format. Only line and arc segments are supported in DXF.

Adjust - adjustment of parameters and cutting modes

Adjustment of wire offsets and cutting modes.

Generate - generate G-code

Generate a G-code program based on the contour and contour adjustments. After successful generation, there is an automatic transition to the G-code editor.

Entry Line - add a new cutline before the first segment

Add a new cutline before the first segment. The cutline is added to the first point of the contour marked with a red square on the plot.

Delete - delete the selected contour from the project

Clear - delete all contours from the project

Set First - Set selected segment the first in the contour

Change Direction - change contour direction

Reverse the order of the segments in the contour. Operation changes the cut direction.

Menu Edit

  • Use as Entry Line

    • Use selected contour segment as the entry line.

  • Rotate

    • Rotate the selected contour by the given angle.

  • Flip Left-Right

    • Flip the selected contour from left to right.

  • Flip Up-Down

    • Flip the selected contour from up to down.

  • Resize

    • Resize the selected contour.
