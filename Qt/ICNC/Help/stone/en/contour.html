Contour Editor

Buttons

Home - go to the Home panel

Load DXF - load DXF file

Загрузить контур реза в формате DXF. Поддерживаются только контуры состоящие из линий и дуг.

Adjust - adjustment of parameters and cutting modes

Adjustment of wire offsets and cutting modes.

Generate - generate G-code

Генерация программы реза в формате G-код на основание контура и заданных параметров реза. При успешной генерации программы происходит автоматический переход в рабочую панель.

Add Entry Line - add a new cutline before the first segment

Линия захода добавляется к первой точке контура, выделенной красным квадратиком на чертеже.

Delete - delete the selected contour from the project

Clear - delete all contours from the project

Set First - to do the selected segment the first in the contour

Change Direction - change contour direction

Reverse the order of the segments in the contour. Operation changes the cut direction.

Menu Edit

  • Use as Entry Line

    • Use the selected contour segment as the entry line.

  • Rotate

    • Rotate the selected contour by the given angle.

  • Flip Left-Right

    • Flip the selected contour from left to right.

  • Flip Up-Down

    • Flip the selected contour from up to down.

  • Resize

    • Resize the selected contour.
