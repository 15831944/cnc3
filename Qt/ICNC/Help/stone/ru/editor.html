Редактор программ G-код

Кнопки

  • Главная - переход на главную панель

  • Новый - создать новый файл

  • Открыть - открыть файл

    Открыть файл с программой работы станка в формате G-код. Расширение у файла должно быть *.NC

  • Сохранить - сохранить текущий файл

  • Сохр. как - сохранить как

    Сохранение программы G-код под новым именем.

  • Чертёж

    Проверка программы и построение контура реза по программе G-код.

  • В контур

    Преобразование программы в формате G-код в контур реза для дальнейшей настройки параметров обработки контура.

  • Работа

    Проверка и преобразование программы в формат ЧПУ, и переход в рабочую панель с текущей программой реза станка.

Поддерживаемые комманды G-код

Управляющий символ

% - разделитель

Ставиться в начале и в конце программы.

Комментарии

({текст комментария})

Пример:
G1 X10 Y0 (линия захода)

G-команды

G0 X{x} Y{y} - быстрое позиционирование

Перемещение в точку (x,y) с максимальной скоростью по линейной траектории. x, y - задаются в мм.

G1 X{x} Y{y} F{speed} - линейная интерполяция

Перемещение в точку (x,y) с заданной скоростью по линейной траектории. x, y - задаются в мм, speed - мм/мин, [0.01, 18].

G2 X{x} Y{y} I{Cx} J{Cy} F{speed} - круговая интерполяция по часовой стрелке

G2 X{x} Y{y} R{radius} F{speed} - круговая интерполяция по часовой стрелке

G3 X{x} Y{y} I{Cx} J{Cy} F{speed} - круговая интерполяция против часовой стрелки

G3 X{x} Y{y} R{radius} F{speed} - круговая интерполяция против часовой стрелки

G4 P{pause} - пауза, мс

Выполнение программы останавливается на указанное время, затем продолжается с того же места.
Пример:
M40 - включения насоса
G4 P30000 - пауза 30 секунд
M82 - включение барабана

G90 - включение режима абсолютных координат (по умолчанию)

G91 - включение режима относительных координат

G92 X{x} Y{y} - смещение абсолютной системы координат


Модальные команды

Модальные команды G1, G2, G3 сохраняют свое действие в следующих за ними кадрах.
Пример:
%
G92 X0 Y0
G1 X10 Y20 F5
X20 Y0
X0
M2
%

М-команды

M0 - технологический останов

Программа ожидает нажатия кнопки «Старт».
Пример:
M40 - включения насоса
G4 P30000 - пауза 30 секунд
M0 - точка останова
M82 - включение барабана

M2, M30 - конец программы

Для завершения программы можно использовать любую из команд.

M40 - включения насоса

M41 - отключение насоса

M82 - включение барабана

M83 - отключение барабана

M84 - включение высокого напряжения

M85 - отключение высокого напряжения

M105 P{value} – скорость вращения барабана

Значения от 1 до 7. 1 - минимальная скорость, 7 - максимальная скорость.
